这次挑战是挑战#1-重复的单词 和 挑战#4-MarkDown 斜体的混合版本。 本次挑战是匹配（立即）重复的 MarkDown 列表项！ 先看看例子吧 - 看例子比解释更容易理解。

重复的项目需要用 MarkDown 强调语法包裹。
第二个 '* List item' 应该被替换为 '* **List item**'

在你输入正则之后，每个测试用例都会标注为是否通过，红色是未通过，绿色是通过，因为是基于 web 的，所以只能使用基于 JavaScript 传统型 NFA (Traditional NFA)正则引擎。它类似于PCRE，但有一些不同之处。
